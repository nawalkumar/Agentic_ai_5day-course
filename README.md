# 🤖 Gemini AI Agents Intensive Architecture Suite

This repository contains my production engineering code built during the Google Developers / Kaggle AI Agents Intensive Course.

## 📁 Repository Structure

- **/my-first-vibe-app** (Day 1): Initial scaffolded workspace deployment structure.
- **/day2-mcp-tools**: Implementation of Model Context Protocol (MCP) servers utilizing Python SDK stdio pipelines.
- **/day3_structured_outputs**: Rigid parsing schemas utilizing Pydantic constraints and Gemini models to enforce clean JSON-RPC interfaces.
- **day4_security.py**: Hard guardrails and interactive Human-in-the-Loop (HITL) gate processing scripts.
- **day5_eval.py**: Integration testing harness and automated runtime dataset assertion metrics.

## 🛠️ Stack Profile
- **Engine Model**: gemini-2.5-flash
- **Runtime Libraries**: `google-genai`, `pydantic`, `mcp`
- **Environment Management**: `uv` fast Python toolchain package registry.
